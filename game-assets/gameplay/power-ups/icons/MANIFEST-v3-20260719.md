# Power-up icons — v3 set (2026-07-19)

## Style contract

- High-detail **pixel art** tech emblem
- Pure **black** background
- Dark gunmetal mechanical frame (hex/oct bezel, purple rivets)
- Neon **cyan** + **amber/orange** + **purple** accents
- Solid fill (not outline-only), centered motif, readable at **32px**
- No text / letters

Anchored on the Citadel Breach trio (`chain-reaction`, `breach-shard`, `echo-charge`) so classic drops match the same family.
